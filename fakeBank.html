What is OSINT?

Open-Source Intelligence
OSINT is the practice of collecting information about a target using only publicly available sources — social media posts, photos, bios, comments, and check-ins. You don't need to hack anything. You just need to look.

Attackers routinely scan Instagram, Facebook, LinkedIn and TikTok profiles before attempting any account takeover. In this activity, Emma's birthday, her pet's name and her hometown were all sitting in plain sight on her public profile — enough to reset her bank password without ever knowing it.
⚠️

The Flaw in Security Questions

Why they're dangerous
Security questions were designed as a backup password — but people treat the answers as public facts. Nobody hides their birthday. Everyone posts about their dog. These facts feel harmless individually, but together they become a master key.

Unlike a real password, a security question answer never changes. Once an attacker finds it on social media, it works forever. This is why many security researchers call security questions one of the weakest links in modern authentication.
🛡️

How to Defend Yourself

Best practices
The most effective defence is simple: lie on security questions.

When a website asks "What is your mother's maiden name?", don't answer truthfully. Treat it like a second password — type something completely random — and store it in a password manager so you don't forget it.
✅ Three habits to start today 1. Use random, unguessable answers for security questions and save them in a password manager.
2. Keep your social media profiles private, and think twice before posting birthdays, pet names, or hometowns.
3. Wherever possible, use a passkey or authenticator app instead of security questions entirely.
